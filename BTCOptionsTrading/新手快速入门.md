## 系统启动

### 1. 启动后端服务

```bash
cd BTCOptionsTrading/backend
python run_api.py
```

等待看到：`Uvicorn running on http://0.0.0.0:8000`

打开新的终端窗口：

```bash

```

## 第二步：查看期权链

在开始交易策略之前，先了解期权市场的基本数据。

### 📊 打开期权链Tab

1. 点击顶部的 **"期权链"** Tab
2. 您会看到一个表格，显示所有可用的期权合约

### 🔍 理解期权链表格

期权链表格分为三列：

```
┌──────────────┬──────────────┬──────────────┐
│   看涨期权   │   执行价格   │   看跌期权   │
│   (Call)     │   (Strike)   │   (Put)      │
├──────────────┼──────────────┼──────────────┤
│ 价格 | IV    │   $50,000    │ 价格 | IV    │
│ 价格 | IV    │   $51,000    │ 价格 | IV    │
│ 价格 | IV    │   $52,000    │ 价格 | IV    │
└──────────────┴──────────────┴──────────────┘
```

### 📖 关键概念

**执行价格（Strike Price）**
- 期权合约约定的买卖价格
- 蓝色高亮的是 **ATM（平值）** 期权，最接近当前BTC价格

**看涨期权（Call）**
- 绿色显示
- 预期价格上涨时买入
- 价格越高，期权越贵

**看跌期权（Put）**
- 红色显示
- 预期价格下跌时买入
- 价格越低，期权越贵

**隐含波动率（IV）**
- 市场对未来价格波动的预期
- IV越高，期权越贵

### 🎮 试试这些操作

1. **选择到期日**：点击日期选择器，查看不同到期日的期权
2. **查看希腊字母**：勾选"显示更多希腊字母"，了解期权的风险指标
3. **刷新数据**：点击"刷新数据"按钮，获取最新价格

## 第三步：创建您的第一个策略

现在让我们创建一个简单的期权策略！

### 🎯 推荐新手策略：买入看涨期权（Long Call）

这是最简单的策略，适合预期BTC价格上涨时使用。

### 📝 创建步骤

1. **打开策略管理Tab**
   - 点击顶部的 **"策略管理"** Tab

2. **选择策略模板**
   - 在"可用策略模板"区域
   - 找到 **"Long Call - 买入看涨期权"**
   - 点击该卡片

3. **填写策略参数**（暂时使用默认值即可）
   - 策略名称：`我的第一个策略`
   - 执行价格：`52000`（略高于当前价格）
   - 到期日：选择一个未来的日期
   - 数量：`1`（买入1张合约）

4. **保存策略**
   - 点击"创建策略"按钮
   - 您会在策略列表中看到新创建的策略

### 💡 策略说明

**Long Call（买入看涨期权）**
- **适用场景**：预期BTC价格会上涨
- **最大损失**：支付的权利金（有限风险）
- **最大收益**：理论上无限（价格涨得越高，赚得越多）
- **盈亏平衡点**：执行价格 + 权利金

**示例**：
- 当前BTC价格：$50,000
- 买入执行价格$52,000的看涨期权
- 支付权利金：$1,000
- 如果到期时BTC涨到$55,000，您可以赚取：$55,000 - $52,000 - $1,000 = $2,000

## 第四步：运行回测

创建策略后，让我们测试它在历史数据上的表现！

### 🔬 回测步骤

1. **打开回测分析Tab**
   - 点击顶部的 **"回测分析"** Tab

2. **选择策略**
   - 在"策略选择"下拉框中
   - 选择刚才创建的 `我的第一个策略`

3. **设置回测参数**
   ```
   初始资金：$10,000（建议新手使用）
   开始日期：30天前
   结束日期：今天
   ```

4. **运行回测**
   - 点击 **"运行回测"** 按钮
   - 等待几秒钟，系统会模拟历史交易

5. **查看结果**
   - 回测完成后，您会看到：
     - 📈 盈亏曲线图
     - 📊 绩效指标
     - 📋 交易明细

## 第五步：分析结果

回测完成后，让我们理解这些数字的含义。

### 📊 关键绩效指标


#### 1. 总收益率

```
总收益率：+15.5%
```

- **含义**：策略的总体盈利百分比
- **如何判断**：
  - ✅ 正数 = 盈利
  - ❌ 负数 = 亏损
  - 🎯 目标：超过10%就很不错了

#### 2. 夏普比率

```
夏普比率：1.2
```

- **含义**：风险调整后的收益率
- **如何判断**：
  - 🌟 > 2.0：优秀
  - ✅ 1.0-2.0：良好
  - ⚠️ 0.5-1.0：一般
  - ❌ < 0.5：较差

#### 3. 最大回撤

```
最大回撤：-8.3%
```

- **含义**：从最高点到最低点的最大跌幅
- **如何判断**：
  - ✅ < 10%：风险较低
  - ⚠️ 10-20%：中等风险
  - ❌ > 20%：高风险

#### 4. 胜率

```
胜率：65%
```

- **含义**：盈利交易占总交易的比例
- **如何判断**：
  - 🌟 > 60%：很好
  - ✅ 50-60%：不错
  - ⚠️ 40-50%：需要改进
  - ❌ < 40%：策略可能有问题

### 📈 盈亏曲线图

盈亏曲线图显示了策略在整个回测期间的表现：

- **蓝线**：组合总价值（现金 + 期权价值）
- **绿线**：累计盈亏

**理想的曲线**：
- 📈 持续向上：策略稳定盈利
- 📉 大幅波动：策略风险较高
- ➡️ 横盘：策略没有明显优势

### 💡 如何改进策略

如果回测结果不理想，可以尝试：

1. **调整执行价格**
   - 价格上涨预期强：选择更接近当前价格的执行价格
   - 价格上涨预期弱：选择更高的执行价格（更便宜）

2. **调整到期时间**
   - 短期（1-2周）：适合快速波动
   - 中期（1个月）：平衡风险和收益
   - 长期（3个月）：降低时间价值衰减影响

3. **尝试不同策略**
   - 市场横盘：试试铁鹰策略（Iron Condor）
   - 预期大幅波动：试试跨式策略（Straddle）

## 进阶功能

当您熟悉基本操作后，可以探索这些高级功能：

### 🎯 复杂策略

#### 跨式策略（Straddle）
- **适用场景**：预期市场会大幅波动，但不确定方向
- **操作**：同时买入相同执行价的看涨和看跌期权
- **风险**：如果市场横盘，两边都会亏损

#### 铁鹰策略（Iron Condor）
- **适用场景**：预期市场在一定区间内波动
- **操作**：卖出两个价差组合
- **风险**：如果价格突破区间，会有较大损失

#### 蝶式策略（Butterfly）
- **适用场景**：预期价格停留在特定价位
- **操作**：买入两个边缘执行价，卖出两倍中间执行价
- **风险**：收益有限，但风险也有限

### 📊 波动率分析

打开"波动率分析"Tab，您可以：

1. **查看波动率微笑**
   - 了解不同执行价的隐含波动率
   - 发现市场对风险的定价

2. **分析期限结构**
   - 比较不同到期日的波动率
   - 判断市场情绪（恐慌/平静）

3. **历史波动率对比**
   - 比较历史波动率和隐含波动率
   - 判断期权是被高估还是低估

### ⚙️ 系统设置

在"设置"Tab中，您可以：

1. **切换网络模式**
   - 🟡 测试网（Testnet）：用于学习和测试
   - 🟢 主网（Mainnet）：真实交易（需要API密钥）

2. **配置API**
   - 输入Deribit API密钥
   - 获取真实市场数据

3. **调整交易参数**
   - 无风险利率
   - 初始资金
   - 手续费率

## 常见问题

### ❓ 数据是真实的吗？

**默认情况下，系统使用模拟数据**，这对学习和测试非常有用。

如果您想使用真实数据：
1. 注册Deribit账号（testnet.deribit.com）
2. 获取API密钥
3. 在"设置"Tab中配置API
4. 详细步骤请参考：[API配置指南](API_CONFIGURATION_GUIDE.md)

### ❓ 回测结果可靠吗？

回测结果基于历史数据，但有以下限制：

⚠️ **注意事项**：
- 历史表现不代表未来收益
- 模拟数据可能与真实市场有差异
- 未考虑滑点和流动性问题
- 手续费可能与实际不同

✅ **回测的价值**：
- 了解策略的基本逻辑
- 比较不同策略的表现
- 学习期权交易的风险管理

### ❓ 如何选择合适的策略？

根据您的市场预期选择：

| 市场预期 | 推荐策略 | 风险等级 |
|---------|---------|---------|
| 强烈看涨 | Long Call | ⭐ 低 |
| 强烈看跌 | Long Put | ⭐ 低 |
| 大幅波动 | Straddle | ⭐⭐ 中 |
| 区间震荡 | Iron Condor | ⭐⭐⭐ 高 |
| 小幅波动 | Butterfly | ⭐⭐ 中 |

### ❓ 期权会过期吗？

是的！期权有到期日：

- **到期前**：可以随时平仓（卖出）
- **到期时**：
  - ITM（价内）期权：自动行权，获得内在价值
  - OTM（价外）期权：归零，损失全部权利金

**建议**：
- 新手先从短期期权开始（1-2周）
- 密切关注到期日
- 在回测中观察到期对策略的影响

### ❓ 如何控制风险？

期权交易的风险管理建议：

1. **限制单笔投资**
   - 每个策略不超过总资金的10%
   - 新手建议5%以下

2. **设置止损**
   - 当亏损达到权利金的50%时考虑平仓
   - 不要让小亏损变成大亏损

3. **分散投资**
   - 不要把所有资金投入一个策略
   - 尝试不同的执行价和到期日

4. **从小额开始**
   - 先用测试网练习
   - 真实交易从小额开始

### ❓ 系统运行很慢怎么办？

如果系统响应缓慢：

1. **检查系统资源**
   - 访问：http://localhost:8000/health
   - 查看CPU和内存使用情况

2. **重启服务**
   ```bash
   # 停止后端
   Ctrl+C
   
   # 重新启动
   python run_api.py
   ```

3. **清理浏览器缓存**
   - 按 Ctrl+Shift+Delete
   - 清除缓存和Cookie
