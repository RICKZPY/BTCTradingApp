## IV 数据说明

### 什么是隐含波动率（IV）？
- IV 是期权市场对未来价格波动性的预期
- IV 越高，期权价格越贵
- IV 反映市场的恐慌程度和不确定性

### IV 与成本的关系
根据之前的分析：
- **低 IV（40-60%）**：市场平静，期权便宜
- **中等 IV（60-100%）**：正常波动，期权价格适中
- **高 IV（100-150%）**：市场波动，期权价格较高
- **极高 IV（>150%）**：市场恐慌，期权价格极高

## 使用建议

### 1. 监控 IV 水平
- 在手机上查看 API 时，注意 IV 字段
- IV > 100% 时，成本可能偏高
- IV > 150% 时，建议暂停交易或减少数量

### 2. 交易决策
- **低 IV 时期**：适合买入期权（成本低）
- **高 IV 时期**：考虑暂停或减少交易量
- **极高 IV 时期**：避免交易，等待 IV 回落

### 3. 成本控制
建议设置 IV 阈值：
- IV < 80%：正常交易（0.1 BTC）
- IV 80-120%：减半交易（0.05 BTC）
- IV > 120%：暂停交易

## 技术细节

### IV 获取方法
```python
async def get_option_iv(self, instrument_name: str) -> float:
    """获取期权隐含波动率（IV）"""
    url = f"{self.trader.base_url}/public/ticker"
    params = {"instrument_name": instrument_name}
    
    async with aiohttp.ClientSession() as session:
        async with session.get(url, params=params) as resp:
            data = await resp.json()
            if 'result' in data and 'mark_iv' in data['result']:
                return data['result']['mark_iv']
    return 0.0
```
