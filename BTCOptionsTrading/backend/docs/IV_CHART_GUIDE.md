# IV 历史图表使用指南

## 访问地址

### 电脑端
```

```

### 手机端
在手机浏览器中访问：
```
http://47.86.62.200:5004/iv-chart
```

建议添加到主屏幕以便快速访问。

## 功能说明

### 1. 统计卡片
页面顶部显示四个关键指标：
- **当前 IV**：最近一次交易的 IV 值
- **平均 IV**：所有交易的平均 IV
- **最高 IV**：历史最高 IV
- **最低 IV**：历史最低 IV

### 2. 双轴折线图
- **蓝色线（左轴）**：IV 变化趋势（%）
- **红色线（右轴）**：交易成本变化（$）
- 可以直观看出 IV 和成本的相关性

### 3. 交互功能
- 鼠标悬停（或手指点击）查看详细信息
- 显示具体时间、IV 值、成本和触发的新闻
- 自动每 30 秒刷新数据

## 图表解读

### IV 与成本的关系
从图表中可以看出：
- IV 上升时，成本通常也上升
- IV 下降时，成本通常也下降
- 两条线的走势高度相关

### 示例分析
假设图表显示：
```
时间        IV      成本
12日 19:00  50%    $570
12日 21:00  90%    $1,288
12日 23:00  140%   $2,397
13日 01:00  200%   $3,325
```

可以看出：
- IV 从 50% 涨到 200%（4倍）
- 成本从 $570 涨到 $3,325（5.8倍）
- IV 每增加 50%，成本大约翻倍

### 交易时机判断
根据图表：
- **IV 在低位（<80%）**：适合买入期权
- **IV 在中位（80-120%）**：谨慎交易
- **IV 在高位（>120%）**：避免交易或减少数量

## 数据说明

### 数据来源
- 从交易日志中解析 IV 数据
- 只显示包含 IV 的交易记录
- 最多显示最近 10 条记录

### 数据更新
- 每次 cron 执行交易后自动更新
- 页面每 30 秒自动刷新
- 手动刷新：重新加载页面

### 历史数据
- 旧的交易记录（3月12-13日早期）没有 IV 数据
- 从添加 IV 追踪功能后的交易开始显示
- 随着时间推移，数据会越来越丰富

## 使用场景

### 1. 交易前检查
在交易前查看当前 IV 水平：
- 如果 IV 处于历史高位，考虑暂停交易
- 如果 IV 处于历史低位，可以正常交易

### 2. 成本分析
对比不同时间的交易成本：
- 找出成本最低的交易时机
- 分析 IV 对成本的影响
- 优化交易策略

### 3. 趋势预测
观察 IV 的变化趋势：
- IV 持续上升：市场恐慌加剧
- IV 持续下降：市场恐慌缓解
- IV 波动剧烈：市场不确定性高

## 故障排查

### 显示"暂无 IV 数据"
原因：
- 还没有包含 IV 的交易记录
- 等待下次 cron 执行

解决：
- 等待下次交易执行（每小时整点）
- 或手动触发一次交易测试

### 图表不显示
原因：
- 网络问题
- Chart.js 加载失败

解决：
- 检查网络连接
- 刷新页面重试
- 查看浏览器控制台错误信息

### 数据不更新
原因：
- 自动刷新失败
- API 服务异常

解决：
- 手动刷新页面
- 检查 API 服务状态
- 查看服务器日志
