## Keyboard Shortcuts

- **Tab**: Navigate between form fields
- **Enter**: Submit form (when focused on input)
- **Escape**: Close modal dialogs

## Troubleshooting

### Toggle not working?
- Check if backend API is running (http://localhost:8000)
- Look for error messages in toast notifications
- Check browser console for errors

### Configuration not saving?
- Verify API key and secret are entered
- Check if .env file has write permissions
- Restart backend API after saving

### System info not updating?
- Refresh the page
- Check network connection
- Verify backend API is responding
